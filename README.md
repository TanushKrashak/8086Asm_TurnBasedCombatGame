# Assembler's Struggle v0.2 – Open Beta

Welcome to the **Open Beta of Assembler's Struggle!**  
Follow the instructions below to set up and run the game on your system.

Assembler's Struggle is a turn-based combat game, allowing 4 players to pair up in teams of 2 and battle each other. With multiple classes and randomised arenas, each game offers a unique combat experience

---

## Installation & Setup

### 1. Extract the Game Files
Unzip the downloaded archive to a preferred directory.

### 2. Install and Configure EMU8086
Launch **EMU8086**.

### 3. Running the Game

1. In **EMU8086**, click **OPEN**.
2. Select `main.asm` from the extracted files.
3. Click **ASSEMBLE** to compile the assembly code into an executable.
4. Click **RUN** once the assembly process is complete.
5. Press **F9** or click **RUN ALL** to start the game.

---

## Troubleshooting

- Ensure that **EMU8086** is properly installed and configured.
- If you encounter errors, verify that the necessary game files are present and extracted correctly.
